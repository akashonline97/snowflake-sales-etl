# Snowflake Sales ETL Pipeline

End-to-end data pipeline that ingests raw sales data, cleans it using Python/PySpark, and loads curated data into Snowflake for analytics.

## Tech Stack

- Snowflake
- Python
- PySpark
- SQL

## Project Architecture

Raw CSV → Python/PySpark → Snowflake Stage → Warehouse Tables → Analytics
